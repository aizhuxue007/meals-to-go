# Meals To Go

Order food, find restuarants all on your phone.

Built with React Native, which runs on both iOS and Android.


## How to Install
1. npm i yarn -g
2. git clone 
3. in working directory 'run yarn'
4. yarn start
5. QR scan or run on iOS or Android emulators
